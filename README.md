A .NET 8 console application that reads ice cream parlor sales data
and generates the following reports:

- Total sales of the store
- Month-wise sales totals
- Most popular item per month with min, max, avg
- Items generating most revenue per month
- Month-to-month growth per item in percentage
- Data inconsistency detection

## Tech Stack
- .NET 8
- C#
- Basic data structures only (Dictionary, List, HashSet)
